Example:  fairfield_county

Files:
fairfield_county.txt - this file
fairfield_county.query.json - JSON query document
fairfield_county.results.json - the results in pretty-printed JSON

Sample usage (includes piping output to a JSON pretty-print program):

$ curl -s https://search.idigbio.org/idigbio/records/_search -d@fairfield_county.query.json | json_pp


Searches for specimen records in Fairfield County.

Does not filter by state our country.



Results file generated by:

$ curl -s https://search.idigbio.org/idigbio/records/_search?size=1000 -d@fairfield_county.query.json | json_pp > fairfield_county.results.json
